# Simulação de Grover

Projeto exemplo para simular o algoritmo de Grover.

Estrutura do projeto:

- `grover_algorithm.py` — implementação básica do algoritmo.
- `grover_advanced.py` — versão avançada / otimizada.
- `grover_interactive.py` — interface interativa (ainda em desenvolvimento).
- `docs/` — imagens e comparações de resultados.
- `exemplos/` — exemplos e notebooks.

Requisitos:

- Python 3.8+
- Ver `requirements.txt` para bibliotecas necessárias

Uso rápido:

```bash
python grover_interactive.py
```
