# Roadmap Fase 4

## Objetivo
Evoluir o simulador para uma plataforma de gestão de casos forenses com persistência, relacionamento entre artefatos e exportação estruturada.

## Entregáveis
- Camada de persistência SQLite com schema versionado.
- Modelos separados para cada entidade forense.
- Grafo investigativo para relações entre entidades.
- Exportadores JSON e STIX 2.1 iniciais.
- Relatório template para casos.
- Testes automatizados.

## Status
- Persistência: implementada.
- Modelos: implementados.
- Grafo: implementado.
- Exportação: implementada.
